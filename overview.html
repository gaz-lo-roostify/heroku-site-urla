Roostify URLA API Overview

Fannie Mae's Form 1003 Uniform Residential Loan Application (URLA) has been in use for more than twenty years. It is the standard form used to apply for a residential loan application, also known as a mortgage. Fannie Mae is coming out with a revised URLA in 2020, with many enhancements to the borrower experience.

The revised URLA is a completely reimagined loan application form that moves the audience focus away from the lender on to the borrower. Because more information must be collected, Roostify has completely refreshed its digital loan application UX to meet the new mandated data requirements.

Roostify has not only built a new application UX but a completely new V2 loan format to support it. Clients are able to turn on this new format in test accounts so they can thoroughly prepare for switching over their production accounts to the new format. Once they switch, all new loans must use the new format. The benefit of Roostify creating a new loan format is it can now support both simultaneously in the same account. This eliminates the need for forced migration of loans in-flight to the new format, lowering the risk of issues arising.
