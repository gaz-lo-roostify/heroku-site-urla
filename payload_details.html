Get Loan Application Payload

To retrieve the payload of submitted loan application use the following command

GET (example): https://qa.roostify.com/api/v1/loan_applications/{find_id}


You need credentials to perform the operation, using a global admin email id

On a terminal, run the following commands:

User.find_by(email:'{email id}').account.token

User.find_by(email:'{email-id}').account.services.map(&:access_token)
